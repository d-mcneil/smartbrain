users table

from the video-ish:
CREATE TABLE users (id SERIAL NOT NULL PRIMARY KEY, name VARCHAR(100) NOT NULL, email VARCHAR(100) UNIQUE NOT NULL, score INTEGER DEFAULT 0, joined TIMESTAMP NOT NULL);

i thought it made more sense to take one id from the other table, so i got rid of the serialization - also added time zone to time stamp
THIS IS THE ONE I USED IN THE APP:
CREATE TABLE users (id INTEGER NOT NULL PRIMARY KEY, name VARCHAR(100) NOT NULL, email VARCHAR(100) UNIQUE NOT NULL, score INTEGER DEFAULT 0, joined TIMESTAMPTZ NOT NULL);
